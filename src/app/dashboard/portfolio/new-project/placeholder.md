## O que é um Portfólio de Projetos?

Um portfólio de projetos não é apenas uma lista de projetos em andamento. É uma coleção estratégica de projetos e programas que são gerenciados de forma centralizada para alcançar objetivos de negócio. Esta seção pode aprofundar essa definição.

- **Projetos Alinhados:** Focar em como garantir que cada projeto contribua para a visão geral.
- **Gestão de Recursos:** Discutir a alocação eficiente de recursos entre projetos concorrentes.
- **Mitigação de Riscos:** Abordar a identificação e gestão proativa de riscos em nível de portfólio.
